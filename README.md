# stayct-web

Independent Next.js 15 foundation for the STAYCT public website.

## Included

- App Router
- TypeScript
- Tailwind CSS setup
- ESLint setup
- Route skeleton
- Metadata architecture
- robots.txt and sitemap generation
- 404, error, and loading states
- Environment config
- Path aliases
- SEO utilities

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
- `npm run typecheck`

## Environment

Required in production:

- `NEXT_PUBLIC_SITE_URL`

Optional for future public API integration:

- `NEXT_PUBLIC_API_BASE_URL`
- `API_BASE_URL`
